# Desafio Detective Quest - Estruturas de Dados e Investigação

O programa simula um jogo de detetive, onde você pode explorar o cenário seguindo os passos
e as orientações do menu inicial.

- O jogo inicia no 'Hall de Entrada', você deve escolher uma das opções e pressionar Enter.
- O programa irá indicar a sua localização atual e as opções que você pode seguir a partir dali.
- O programa encerra quando você chega a um cômodo sem saída.
- Por fim, a opção '0' possibilita ao usuário sair do jogo quando desejado.
